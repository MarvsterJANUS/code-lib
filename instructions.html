How to Use

Get code snippets into your project in 3 easy steps

1

Browse & Find

Open the Library and browse components by category (Buttons, Cards, Forms, etc.) or use the search bar to find what you need.

2

Preview & Copy

Click "View Code" to see the HTML/CSS code, then click "Copy Code" to copy it to your clipboard.

3

Paste & Customize

Paste the code into your HTML file. Customize the colors, text, and styles to match your project's design.

Quick Tips

  • Each component is self-contained - just paste and it works
  • Use the grid/list toggle to change how you view components
  • Switch between dark and light themes with the sun/moon button
  • All code is free to use in any project
